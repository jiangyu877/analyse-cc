# User Guide

Administrators assign roles; operators manage orders, payments, refunds and approvals; analysts run reports and models. The commerce flow is order, payment, refund request, approval, and inventory reconciliation.

Upload knowledge documents, publish approved versions, then ask QA questions. Answers show citations and retrieval evidence; feedback can create support tickets.

Reports support customer and sales views, export, and ADS refresh. Model results are persisted by task: RFM (recency/frequency/monetary value), KMeans segments, churn risk, amount forecasting, sales forecasting, and product recommendations. Review sample size and confidence before acting on predictions.
